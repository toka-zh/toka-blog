# 排序

|              | 时间复杂度          | 稳定性 | 特点                       |
| :----------- | ------------------- | ------ | -------------------------- |
| 直接插入排序 | n*n                 | 稳定   | 顺序存储和链式存储的线性表 |
| 折半插入排序 | n*log_2 n           |        | 顺序存储的线性表           |
| 希尔排序     | n^1.3 最坏情况下n*n | 不稳定 | 顺序存储的线性表           |

|              | 时间复杂度（平均） | 最好    | 最坏 | 稳定性 | 特点         |
| :----------- | ------------------ | ------- | ---- | ------ | ------------ |
| 冒泡排序     | O(n*n)             |         |      | 稳定   | 简单粗暴     |
| 快速排序     | n*log_2 n          | log_2 n | n*n  | 不稳定 | 效率最优！！ |
| 简单选择排序 | O(n*n)             |         |      |        |              |
| 堆排序       | O(n*log_2 n)       |         |      |        |              |
